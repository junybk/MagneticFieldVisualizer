# Magnetic Field Visualizer (Magneto Lab)

`three.js` を使って、棒磁石/U字磁石まわりの磁場を 3D で可視化するシングルファイル Web アプリです。  
磁力線表示、粒子表示、方位磁針プローブ、2磁石間のリアルタイム計測を使って、磁場の重ね合わせや相互作用を直感的に観察できます。

## 特徴

- 棒磁石・U字磁石を追加し、ドラッグで配置/移動
- Shift+ドラッグで磁石を回転
- 磁力線/粒子/両方の可視化モード切り替え
- 磁力線密度・磁石強度・アニメーション速度をスライダーで調整
- 方位磁針プローブを任意位置に置いて磁場方向を確認
- 2つの計測対象磁石に対して、距離・中央点の磁場強度・引力/反発判定を表示
- FPS、磁石数、磁力線本数、モードをフッターに表示

## 使い方

### 1) アプリを起動

ビルド工程は不要です。静的ファイルとして配信してブラウザで開きます。

#### Python がある場合

```bash
python -m http.server 8000
```

その後、ブラウザで以下を開きます。

- [http://localhost:8000/index.html](http://localhost:8000/index.html)

#### Node.js がある場合（任意）

```bash
npx serve .
```

表示されたローカルURLで `index.html` を開いてください。

### 2) 基本操作

- **磁石を移動**: ドラッグ
- **磁石を回転**: Shift + ドラッグ
- **視点回転**: 右クリックドラッグ
- **パン**: 中クリックドラッグ / Space + ドラッグ
- **ズーム**: マウスホイール
- **視点リセット**: 左パネルの `↻ 視点をリセット`

### 3) UI操作

- 左パネルで磁石/方位磁針の追加・削除（クリア）
- 可視化モードを `磁力線 / 粒子 / 両方` で切り替え
- `磁力線の密度`、`磁石の強さ`、`アニメーション` を調整
- 計測対象は左パネルの磁石項目のチェックで2つ選択

## 実装メモ

- 磁場計算は双極子近似（各磁石を N/S の点磁荷として計算）
- 初期配置として磁石2つと方位磁針1つを自動生成
- 画面幅が狭いときは左サイドバーがドロワー化（モバイル対応）

## 注意点

- `three.js` とフォントを CDN から取得するため、初回表示時はインターネット接続が必要です
- `package.json` / テスト設定 / CI 設定は現在含まれていません
- 永続化（保存/読み込み）機能はなく、状態はページ再読み込みで初期化されます
